Question 5

Il existe plusieurs manières de naviguer et afficher les fichiers.
La commande "ls ./toto/titi" nous permet de voir les fichiers de titi, lorsqu'on est dans le directory parent de toto.
La commande "ls /toto/tiit" nous permet de voir les fichiers de titi depuis la racine.
La commande "ls ../toto/titi"  nous permet de voir les fichiers de titi, depuis un repertoire frère de toto.
